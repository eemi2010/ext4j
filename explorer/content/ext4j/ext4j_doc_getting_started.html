Getting started with Ext4j

The following software components are required in order to be able to develop applications using Ext4j.

- The Ext JS JavaScript library (4.1 or higher)
- The Google Web Toolkit (GWT) SDK (2.4 or higher)
- The Java Development Kit (1.6 or higher)
- The Ext4j jar file
And for a better developement experience we recommend:
- An Java IDE of choice with the Google Web Toolkit plugin installed

The following steps will walk you through creating a simple Hello World app with Ext4j. We will be using Eclipse with the Google Plugin for Eclipse.Let s get started.

- 1) Create a new Web Application Project (GWT) project in your IDE (We will be using Eclipse with the Google Plugin for Eclipse). Enter your project's details as appropriate.

- 2) Add the ext4j-xxx.jar file to the project's classpath.

- 3) Download the Ext JS library and copy it into the war folder of your project

- 4) Modifiy the index.html file of your project to reference the Ext JS library


Note that we are using the gray theme here.

- 5) Edit the .gwt.xml file of your project to inherit the Ext4j Module

- 6) Modify the EntryPoint of your GWT application to extend com.eemi.ext4j.client.core.ExtEntryPoint
