Requirements

The following software components are required in order to be able to develop applications using Ext4j.

- The Ext JS JavaScript library (4.1 or higher)
- The Google Web Toolkit (GWT) SDK (2.4 or higher)
- The Java Development Kit (1.6 or higher)
- The Ext4j jar file
And for a better developement experience we recommend:
- An Java IDE of choice with the Google Web Toolkit plugin installed
